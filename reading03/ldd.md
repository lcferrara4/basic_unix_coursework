TLDR - ldd
==========

Overview
--------

ldd prints the shared libraries required by a program.

The syntax is ldd [option]... FILE...

Examples
--------

- General use case: ldd /bin/ls

- For verbosity: ldd -v /bin/ls

- To print unused direct dependencies: ldd -u /bin/ls
